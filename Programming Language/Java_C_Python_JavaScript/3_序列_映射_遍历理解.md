## 遍历[ : / in / of ]的理解

```
# js
in > 提及一个对象(映射)，in需要通过**主键寻找**内部的元素 > for key in object | for index in array
of > 提及一个数组(序列)，of指整体中的一个，属于整体**包含个体**的关系 > for item of array

# java
: > 提及一个序列,:指整体中的一个，属于整体**包含个体**的关系 > for item : items

# python
in > 简化思路，dict需要**主键**去寻找元素，list需要**个体** > for item in list_ | for key in dict_

# PS:
- 主键寻找
- 包含个体
```
