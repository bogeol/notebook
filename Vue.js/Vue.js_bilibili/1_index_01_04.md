- **Data Bindings** => :
- **DOM Listeners** => @

```
- index_01:
  - new Vue({})
  - Vue -> {{}} [**tag content**] [Vue directives+mustache] [**Data Bindings**]
    - new Vue({data}) -> {{variable}}
  - JavaScript Data Types
    - Primitive Types
    - Non-Primitive Types [Object]
- index_02:
  - Vue -> : / @ [**tag attributes**] [v-bind: == : / v-on: == @] [MVVM(View - ViewModel - Model)]
    - new Vue({data}) -> :attribute="variable" [**Data Bindings**]
    - new Vue({methods}) -> @event="function" [**DOM Listeners**]
- index_03:
  - Vue -> Life Cycle
    - new Vue({created})
    - ...
- index_04:
  - Vue
```
